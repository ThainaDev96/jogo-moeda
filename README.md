# Jogo da moeda
Catch the Coin é um jogo 2D desenvolvido com Pygame onde o jogador controla um barco para coletar moedas que caem do céu e levá-las ao porto antes que o peso excessivo o faça afundar.
O jogo possui sistema de fases com dificuldade progressiva, vidas, itens que causam dano ou cura, animações, efeitos sonoros e interface com menu interativo e tela de instruções.
